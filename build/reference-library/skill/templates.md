## Skill Starter Template

```markdown
---
name: {{SKILL_NAME}}
description: >
  {{SHORT_DESCRIPTION}}.
  Use when: {{ACTIVATION_TRIGGERS}}.
license: MIT
---

# {{DISPLAY_NAME}}

## Overview
{{WHAT_THIS_SKILL_DOES}}

## Instructions
{{DETAILED_GUIDANCE}}
```

**Placeholders:**
- `{{SKILL_NAME}}`: 1-64 chars, lowercase + hyphens
- `{{SHORT_DESCRIPTION}}`: 250-350 characters
- Keep total SKILL.md under 5000 tokens
